
# DSA 210 - Data Science Project

## Project Overview
This project is part of the DSA 210 course. The goal is to apply data science techniques to a real-world dataset.

## Planned Steps
- Data collection
- Data cleaning & preprocessing
- Exploratory Data Analysis (EDA)
- Hypothesis testing
- Machine Learning modeling
- Evaluation and interpretation

## Repository Structure
- data/ → datasets
- notebooks/ → Jupyter notebooks
- src/ → Python scripts
- results/ → outputs, figures
